## Inquiry Matching Guide

| Client State | Recommended Assessments |
|--------------|------------------------|
| New client / intake | DASS-21, ASRS, values (PVQ or VIA) |
| ADHD/EF concerns | BDEFS, ASRS, MAI |
| Emotion dysregulation | DERS, Self-Compassion, DASS-21 |
| Stuck in therapy | AAQ-II, WAI-SR, reflection rubrics |
| Learning/growth focus | MAI, MSLQ, Kolb |
| Values clarification | PVQ, VIA |
| Self-criticism | Self-Compassion Scale |

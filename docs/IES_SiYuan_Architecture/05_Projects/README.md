# 05_Projects

Each subfolder under `05_Projects` is a single project.

Recommended internal structure for a project:

- `Goals.md`
- `Questions.md`
- `Research.md`
- `Maps.md`
- `Decisions.md`
- `Plan.md`
- `Status.md`
- `Logs.md`

See `/07_System/Templates/Template_Project_Structure.md` for a canonical scaffold.

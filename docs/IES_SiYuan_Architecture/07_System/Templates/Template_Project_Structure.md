# TEMPLATE: Project Structure

> Recommended skeleton for each project under `/05_Projects`.

## Folder Layout

- `README.md`
- `Goals.md`
- `Questions.md`
- `Research.md`
- `Maps.md`
- `Decisions.md`
- `Plan.md`
- `Status.md`
- `Logs.md`

---

### README.md

- Short description of the project.
- Why it matters.
- Links to key maps and concepts.

### Goals.md

- Outcomes you care about.
- Constraints.
- Success criteria.

### Questions.md

- Open questions / unknowns.
- Hypotheses.

### Research.md

- Links, notes, summaries from external sources.
- References to Seedlings and Concept pages.

### Maps.md

- Flow Maps specifically relevant to this project.
- Embeds or references to `03_Flow_Maps`.

### Decisions.md

- A list of decision-blocks.
- Each with status, rationale, and links.

### Plan.md

- High-level roadmap.
- Phases or milestones.

### Status.md

- Current state.
- Recently completed.
- Risks / blockers.

### Logs.md

- Chronological log-entry-blocks for changes and notable events.

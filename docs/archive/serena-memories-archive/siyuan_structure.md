## Document Types

### Session Documents
Location: `/{Notebook}/Sessions/YYYY-MM-DD-{topic}`
- Record of exploration session
- Topics explored, entities created, connections
- Links to transcript and entities

### Entity Pages
Location: `/{Notebook}/Entities/{name}` or by type subfolder
- Individual concepts (personal or literature)
- Evolve over time, accumulate quotes
- Block attributes for metadata

### Integrated Theories
Location: `/Therapy/Integrated/{track-name}`
- Synthesized understanding
- Built from entities
- Tracks: 1-Human Mind, 2-Change Process, 3-Method

### Transcripts
Location: `/{Notebook}/Sessions/Transcripts/YYYY-MM-DD-{topic}`
- Raw conversation preserved
- AI processing notes

## Key Pages

- **Knowledge Hub** - Dashboard with stats, recent, suggestions
- **Entity Search** - Search interface for knowledgebase
